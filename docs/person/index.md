# 初始化角色

> 一个角色刚刚创建，可能需要用到这些

## 升级

```
.levelup 79
```

正数增加，负数减少

## 增加可用天赋点

:::warning
这个命令是临时的，下线之后在上就重置了，永久存在需要改倍率
:::

```
.modify talentpoints 159
```

## 加钱

```
.modify money 90000000
```

以铜币为单位，别超过这个数字(9000 金币)，不然容易出 bug，只能大退

## 36 格包

```
.add 23162 4
```

这个 wlk 版本最大的了， 后面的 4 是个数 要几个
