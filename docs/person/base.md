# 基础命令

## 加速

```
.modify speed 1
```

1 是默认速度，10 是 10 倍速度

## 完成任务

```
.quest complete 任务名
```

回车后粘贴命令，shift + 鼠标左键点下任务名字，就可以自动添加上任务名

## 复活
```
.revive
```
可以做个宏
```
/1 .revive
```


## 重置冷却
```
/1 .cooldown
```

.appear                    移动到目标位置。
.summon                    召唤目标到你的位置。
