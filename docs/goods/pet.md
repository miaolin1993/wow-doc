## 猎人稀有宝宝

这个命令是把宝宝召唤到脸上，需要手动抓

```
# 洛卡纳哈（索查拉盆地的豹子）
.npc add 32517

# 古德利亚（祖达克的紫色老虎）
.npc add 33776

# 逐日（电狼）
.npc add 35189

# 阿克图瑞斯（熊）
.npc add 38453

```
